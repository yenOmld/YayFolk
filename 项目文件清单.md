## 🚀 使用步骤

### 1. 克隆到新仓库
```bash
cd add
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-new-repo-url>
git push -u origin main
```

### 2. 配置必要信息
编辑 `travelate-backend/src/main/resources/application.yml`：
- 填写 GitHub OAuth 配置
- 填写数据库密码
- 填写邮箱配置
- 填写第三方 API 密钥

编辑 `travelate-backend/src/main/resources/application.properties`：
- 确认数据库连接信息

### 3. 初始化数据库
```bash
mysql -u root -p < travelate_schema.sql
mysql -u root -p < travelate_data.sql
mysql -u root -p < landmark_tabel.sql
mysql -u root -p < attractions_images.sql
```

### 4. 启动后端
```bash
cd travelate-backend
mvn spring-boot:run
```

### 5. 启动前端
```bash
cd travelate-vue
npm install
npm run dev
```

## 📝 注意事项

1. **敏感信息**：所有配置文件中的占位符（如 `YOUR_XXX`）都需要替换为实际值
2. **数据库密码**：务必修改默认密码，使用强密码
3. **第三方服务**：需要自行申请百度、腾讯、阿里云等服务的 API 密钥
4. **Redis**：确保安装并启动 Redis 服务（默认端口 6379）
5. **地标识别**：如需使用地标识别功能，需要部署模型服务（默认端口 5000）
