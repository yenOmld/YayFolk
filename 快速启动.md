# YayFolk 快速启动指南

## 🚀 快速启动（4 步搞定）

### 第一步：拉取项目

#### 1. 克隆仓库
```bash
git clone https://github.com/yenOmld/YayFolk.git
cd YayFolk
```

---

### 第二步：安装依赖

#### 1. 后端依赖
```bash
cd travelate-backend
mvn install
```

#### 2. 前端依赖
```bash
cd travelate-vue
npm install
```

---

### 第三步：配置数据库

#### 1. 创建 MySQL 数据库
```sql
CREATE DATABASE travelate CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

#### 2. 导入数据
依次执行以下 SQL 文件：
```bash
# 在 MySQL 中执行
mysql -u root -p travelate < travelate_schema.sql
mysql -u root -p travelate < travelate_data.sql
```

#### 3. 配置数据库连接
编辑 `travelate-backend/src/main/resources/application.properties`：
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/travelate?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
spring.datasource.username=root
spring.datasource.password=你的数据库密码
```

---

### 第四步：启动服务

#### 1. 启动后端
```bash
cd travelate-backend
mvn spring-boot:run
```
后端默认运行在：`http://localhost:8080`

#### 2. 启动前端（新终端）
```bash
cd travelate-vue
npm run dev
```
前端默认运行在：`http://localhost:5173`

---

## ✅ 完成！

访问 `http://localhost:5173` 即可使用应用。

管理员后台登录页：`http://localhost:5173/admin-login`

## 📝 配置说明（可选）

### 最小化配置

如果只想快速测试，**只需配置数据库**，其他配置可以保持默认（不会报错）：

#### application.yml 中的可选配置
以下配置**不填也能运行**，只是对应功能无法使用：

```yaml
# GitHub OAuth - 不填影响不大，只用账号密码登录即可
GITHUB_CLIENT_ID: YOUR_GITHUB_CLIENT_ID
GITHUB_CLIENT_SECRET: YOUR_GITHUB_CLIENT_SECRET

# 邮箱验证码 - 不填无法使用邮箱注册，但可以用用户名密码注册
mail:
  username: YOUR_EMAIL@163.com
  password: YOUR_EMAIL_PASSWORD

# 百度 AI/翻译 - 不填无法使用 OCR 和翻译功能
baidu:
  ai:
    app-id: YOUR_BAIDU_APP_ID
    api-key: YOUR_BAIDU_API_KEY

# 阿里云 OSS - 不填无法上传图片，但不影响其他功能
aliyun:
  oss:
    access-key-id: YOUR_ALIYUN_ACCESS_KEY_ID
    access-key-secret: YOUR_ALIYUN_ACCESS_KEY_SECRET
```

### Redis（可选）
Redis 用于存储验证码，如果不用邮箱注册功能，**可以不安装 Redis**。

## 🔧 常见问题

### 1. 后端启动失败
- 检查 MySQL 是否运行
- 检查数据库密码是否正确
- 检查端口 8080 是否被占用

### 2. 前端无法连接后端
- 确保后端已启动
- 检查 `vite.config.js` 中的代理配置

### 3. 登录注册问题
- 确保数据库表已正确创建
- 检查数据库连接配置
