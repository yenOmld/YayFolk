YayFolk（有戏）——非遗社群

本项目是**面向国内外用户的非遗文化社群平台**，集非遗展示、传承人（商家）服务、AI 文化路线规划、打卡成就、跨境交流于一体。平台支持**多语言****国际化**与**内容实时翻译**，用户可体验非遗文化、购买作品、参加活动、组团搭子；传承人可认证开店、发布商品与活动；管理员通过 AI + 人工审核保障内容安全，整体实现传统文化的数字化、国际化传播。

# 一、整体流程化思路（项目全貌）

1. ## 项目定位

**非遗文化国际社群 + 传承人社群 + 体验打卡 + AI 智能路线 + 电商 / 活动**面向：国内外用户、非遗传承人（商家）、平台管理员

1. ## 三大角色

1. **普通用户**看非遗、看帖子、买商品、参加活动、打卡成就、AI 路线规划、分享
2. **商家（非遗传承人）**认证入驻 → 发商品 → 发活动 → 发教学 → 发帖子 → 私信粉丝
3. **平台****管理员**审核商家、审核帖子、AI 审核辅助、官方发非遗内容、客服、数据管理

1. ## 核心业务流程（最关键）

### 用户流程

注册 / 登录 → 浏览非遗介绍 → 看社群帖子 → 找商家 → 报名活动 → 打卡成就 → AI 生成文化路线 → 收藏分享

### 商家流程

申请认证 → 管理员审核 → 发布活动 / 帖子 → 接待用户预约 → 核销  → 获得评价

### 管理员流程

审核商家 → 审核帖子（AI + 人工） → 发布官方非遗内容 → 处理客服 → 管理违规内容

1. ## 内容体系

1. **官方非遗库**（所有非遗介绍、列表）
2. **用户帖子**（分享、非遗识别自动打标签）
3. **商家帖子**（带商品 / 活动链接）
4. **组团搭子帖**（找人一起体验）
5. **AI 路线**（朝代 / 地域 / 品类 / 诗词 / 节气路线）

1. ## 交易与活动体系

- 商品订单：待付款 → 待发货 → 待签收 → 完成
- 活动预约：待预约 → 预约成功 → 到店核销 → 完成
- 打卡成就：体验服务 → 自动打卡 → 获得成就
