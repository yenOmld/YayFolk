# YayFolk Local Classifier Service

Run the local classifier service from this directory:

```bat
start.bat
```

Defaults:
- Host: `127.0.0.1`
- Port: `5001`
- Checkpoint: `D:/Cursor/program/pic/outputs/mobilenetv2_best.pth`

Optional environment variables:
- `YAYFOLK_CLASSIFIER_HOST`
- `YAYFOLK_CLASSIFIER_PORT`
- `YAYFOLK_CLASSIFIER_CHECKPOINT`
- `YAYFOLK_CLASSIFIER_TOPK`
